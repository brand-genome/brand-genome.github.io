Definition: {{ .Title }}

{{ if eq $taxonomy "lightings" }}

Lighting describes the illumination style used in a brand's visuals, such as bright, moody, natural, or studio-lit scenes.

It shapes emotional tone and attention focus, influencing how professional, energetic, or intimate a brand appears.

{{ else if eq $taxonomy "perspectives" }}

Perspective describes camera angle and viewpoint, such as eye-level, low-angle, overhead, or close framing.

It affects perceived power, intimacy, and context in brand storytelling.

{{ else if eq $taxonomy "photography_genres" }}

Photography genre captures the image style family, such as portrait, product, lifestyle, documentary, or editorial.

It indicates how brands structure visual narratives for audiences and use-cases.

{{ else if eq $taxonomy "concepts" }}

Concepts represent high-level visual themes, such as innovation, trust, adventure, sustainability, or community.

They summarize the semantic intent encoded in campaign imagery.

{{ else if eq $taxonomy "image_backgrounds" }}

Image background describes scene context behind the subject, such as plain, environmental, textured, or abstract.

Background choices influence focus, realism, and perceived brand setting.

{{ else if eq $taxonomy "color_schemes" }}

Color scheme captures palette harmony structures, including monochrome, complementary, analogous, and high-contrast mixes.

It reflects visual consistency and emotional positioning in brand communication.

{{ else if eq $taxonomy "depths" }}

Depth refers to spatial layering and focus falloff, such as shallow depth-of-field or deep scene clarity.

It controls emphasis and realism by guiding where viewers look first.

{{ else if eq $taxonomy "image_effects" }}

Image effects capture applied stylistic treatments like blur, grain, glow, vignette, and high dynamic contrast.

These effects reinforce mood and brand aesthetics beyond raw photography.

{{ else if eq $taxonomy "hair_styles" }}

Hair style captures how human subjects are visually styled in promotional images.

It contributes to identity cues for demographic targeting and brand personality.

{{ else if eq $taxonomy "facial_expressions" }}

Facial expression captures visible emotional states such as joy, confidence, neutrality, or engagement.

Expression directly influences tone, trust, and perceived brand empathy.

{{ else if eq $taxonomy "clothing_styles" }}

Clothing style describes apparel categories in images, from formal and casual to athletic or costume-like outfits.

It signals context, aspiration level, and audience alignment.

{{ else if eq $taxonomy "clothing_colors" }}

Clothing color captures dominant outfit palette characteristics such as vibrant, muted, neutral, or branded tones.

It supports consistency between human presentation and brand color language.

{{ else if eq $taxonomy "posings" }}

Posing describes body posture and action stance, including standing, seated, active, and expressive gestures.

It shapes dynamism and confidence cues in visual storytelling.

{{ else if eq $taxonomy "gazes" }}

Gaze direction indicates where subjects look, including direct eye contact, off-frame focus, or downward attention.

It affects viewer connection, narrative flow, and intent.

{{ else if eq $taxonomy "body_sections" }}

Body section captures visible framing scope, such as face-only, upper-body, full-body, or cropped composition.

It controls intimacy and contextual detail in communication assets.

{{ end }}
