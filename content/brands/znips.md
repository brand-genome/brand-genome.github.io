## Brand Guidelines

### 2009

**Logo Information:**

- The Znips logo must be clearly visible wherever it is used. It should never be redrawn, modified, or enclosed in a box or frame.
- Logo available in small (≤40mm), medium (>40mm), and large (≥400mm) sizes.
- Freezone rules state that a certain space must be maintained around the logo for better appearance and visibility.
- Incorrect usage: Logos should not be cropped in a way that interferes with readability or intentionally using unauthorized box enclosures.
- Logos can be used as patterns but should follow a two-color combination rule for these patterns.

**Color Information:**

- Pantone 144: C:0 / M:48 / Y:100 / K:0, R:242 / G:141 / B:47 (Primary color)
- Pantone 445: C:5 / M:0 / Y:10 / K:68, R:85 / G:90 / B:87 (Secondary color)
- Pantone Cool Grey 5: C:0 / M:0 / Y:0 / K:29, R:188 / G:191 / B:195 (Secondary color)
- Black: C:0/ M:0 / Y:0 / K:100, R:0 / G:0 / B:0 (Neutral color)

**Typography Information:**

- Helvetica Neue family is the recommended typeface for all printed materials. Weights: Light and Bold.
- Main headings and subheadings should be set in Helvetica Neue Bold, while paragraphs should be set in Helvetica Neue Light.

**Spacing Layout:**

- Freezone or clear-space guidelines dictate maintaining isolation space around the logo for enhanced clarity and visibility.

**Notes:**

- Contact Mind Design for any brand-related queries and exact dimensions for applications like retail signage or decorations.
