Which logo colour type is to be used when?
CMYK (Cyan, Magenta, Yellow, Black) Version:
The CMYK logo is to be used in all print applications where the 4 colour process is being used.
For example: Brochures, press adverts or any commercial 4 colour printing. 
PMS (Pantone Matching System) Version:
The PMS logo is to be used in print mediums where the printing process doesn™t
allow the CMYK colour system but does allow PMS colour system.
For example: embroider y, screen printing, some sign writing.
RGB (Red, Green, Blue) Version:
The RGB version is to be used on all digital and screen based reproductions.
For example: Website, Television Commercial, Powerpoint presentations, as well as o˜ice 
based usage, such as in Microso˚ Word.
MONO Version:
The Mono logo is to be used on all one colour mediums where tones of a
single colour are NOT possible.
For example: One colour printed material such as faxes and other one colour
printing that my be subject to photocopying or low quality reproduction.

Paint brush graphic
The elements of the brush strokes tie in with the first of the regional logo concepts. Brush 
strokes are applied by hand, one stroke at a time, until the picture is complete. This visual 
metaphor is featured in the brand and represents the people of the community, including 
Council, that all work together for Gilgandra.
The brush strokes are used across the region branding. The graphic can be used entering 
from the edge of the page. This would typically be for portrait format collateral i.e. DL flyers. 
For landscape format collateral, the graphic can overlap a white shape on one side and an 
image of the other i.e. Billboards.
