## Brand Guidelines

### 2021

**Logo Information:**

- The Dott logo is constructed using round shapes to create a friendly and legible appearance across applications.
- Clear space around the Dott logo must equal the height of the letter 'd', except for app and social icons.
- The Dott logo is available only in black and white. Color codes are provided for both variations: White - HEX: #FFFFFF / RGB: 255, 255, 255 / CMYK: 0, 0, 0, 0; Black - HEX: #000000 / RGB: 0, 0, 0 / CMYK: 100, 100, 100, 100.
- Illustrations of incorrect logo usage: No rotating, stretching, adding elements, boxing in, outlining, adding drop shadows, using colors, masking with images, or modifying the logo.
- Guidelines for collaborations (vertical logo lockups): Use the height of 'd' as clear space between logos and the divider. The divider line must be longer than the logo, at half the length of 'd'. Partner logo width should match the Dott logo.
- Horizontal logo lockup guidelines: Same as vertical; ensure the height of 'd' is used as clear space, and the divider length half the height of 'd'. Partner logo should match the width of the Dott logo.
