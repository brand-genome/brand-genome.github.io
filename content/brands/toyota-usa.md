## Brand Guidelines

### 2013

**Logo Information:**

- Primary Signature with tagline and Primary Signature without tagline are provided in all templates. Horizontal lockup min. "x" height: 0.0325 inch for print and 5 pixels for digital. Stacked lockup minimum "x" height is also 0.0325 inch for print and 5 pixels for digital.
- Lockups without tagline: Horizontal lockup minimum "x" height is 0.0375 inch for print and 6 pixels for digital. Stacked lockup minimum "x" height matches this requirement.
- Clear space rules for all lockups require equal cap height of the first letter of the sub-brand name. This ensures visual integrity and prevents overlapping elements.
- Examples of improper logo usage include modifying spacing or proportions, altering colors beyond approved versions, adding shadows or patterns, placing logos on insufficiently contrasting backgrounds, creating new lockups outside predefined templates, and creating decorative patterns.

**Color Information:**

- Toyota's official color palette includes Toyota Red (PMS 186C) with CMYK: 00 100 100 10, RGB: 215 25 33, HEX: #D71921; 80% Black (PMS Black C 80% tint) with CMYK: 00 00 00 80, RGB: 88 89 91, HEX: #58595B; White with CMYK: 00 00 00 00, RGB: 255 255 255, HEX: #FFFFFF; and Black with CMYK: 00 00 00 100, RGB: 00 00 00, HEX: #000000.
- Full-color logo usage guidelines specify placement on backgrounds ranging from 100% white to 10% black for significant contrast. Black-and-white versions must be used for backgrounds darker than 10% black.
- Specific rules for lockup placement on colored or photographic backgrounds emphasize maintaining significant contrast and staying within predefined color value ranges (e.g., 10%-100% black or their equivalents). Usage should align with full-color and black-and-white lockup rules.

**Typography Information:**

- Sub-brand names formatting rules: One-word sub-brands are set horizontally, never stacked. Two-word sub-brands require style changes. The first word is Helvetica Neue LT Std 75 Bold in 80% Black, and the second word is Helvetica Neue LT Std 55 Roman in Toyota Red.
- Three-word and four-word sub-brand names incorporate style changes and line breaks. First portions use Helvetica Neue LT Std 75 Bold in 80% black, and second portions use Helvetica Neue LT Std 55 Roman in Toyota Red.

**Imagery Photography:**

- When placing the full-color lockup on photography, ensure background value ranges are between 100% white and 10% black. Use black-and-white versions for backgrounds between 10%-100% black.

**Spacing Layout:**

- Clear space surrounding lockups equals the cap height of the sub-brand name's first letter to ensure attention and prevent overlapping design elements.

**Notes:**

- File formats for branding assets: Adobe Illustrator (AI) for original vector files; EPS for print and signage; PNG for lossless web-based use derived from vector. JPG replaces PNG only for imagery exceeding color requirements.
- File access is restricted via directories such as http://adplanner.toyota.com and http://www.tmsbroadcastorders.com requiring usernames and passwords.
