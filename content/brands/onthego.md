## Brand Guidelines

### 2016

**Logo Information:**

- Corporate Logo: Primary logo version with the Registered Trademark symbol (®). Secondary versions include Black and Reverse for colored backgrounds or images.
- Corporate Icon: Secondary brandmark for corporate contexts. Does not include the Registered Trademark symbol (®) for simplicity. Versions include Black and Reverse for use on colored backgrounds or images.
- Artwork + Garment Logo: Used for co-branded sponsorship/partnership apparel and visual materials. Does not include the Registered Trademark symbol (®). Versions include Primary, Black, and Reverse for colored backgrounds or images.
- Artwork + Garment Icon: Similar applications to the logo but specifically for co-branded garments. Includes Primary, Black, and Reverse versions. Custom colors allowed with prior approval.
- Minimum clear space for the logo is equal to the vertical height of the 'N' in the logo. For the icon, it is 50% of its size. Clear space applies evenly on all four sides.
- Incorrect logo usage includes applying colors outside guidelines, skewing, rotating, adding outlines, or using effects like drop shadows.

**Color Information:**

- Corporate brand colors include OTG Green (CMYK: 55 3 100 0, RGB: 128 188 65, HEX: #81BC41, PMS: 376 C), OTG Gray (CMYK: 0 0 0 80, RGB: 88 89 91, HEX: #58585B, PMS: COOL GRAY 11 C), Black (CMYK: 0 0 0 100, RGB: 0 0 0, HEX: #000000, PMS: BLACK C), and White (CMYK: 0 0 0 0, RGB: 225 225 225, HEX: #FFFFFF).
- Core color range includes tested Pantone colors for consistency in garment sublimation. Designers are advised to use the standard palette.

**Typography Information:**

- Primary Typeface: Roboto Condensed, available in Regular and Bold weights. Bold is used for headings in uppercase, and Regular is used for long copy in sentence case. Roboto Condensed is appropriate for both web and print applications.
- Alternate Typeface: Lato, used only when Roboto Condensed is unavailable.
- Body text should be in OTG Gray or Black. Text can appear in any core brand color.

**Spacing Layout:**

- Logo and Icon must adhere to minimum size and spacing guidelines. Clear space calculation: vertical height of 'N' in Logo or 50% proportionally for Icon.

**Brand Voice:**

- Wordmark usage rules: ONTHEGO® wordmarks must maintain specific formatting, including text case, spacing, and symbols. Acceptable formats: ONTHEGO®, OTG (shorthand), ONTHEGO SPORTS. Unacceptable formats include 'On The Go.'
- Trademark symbol ® must always be used with Corporate Logo and Icon in marketing or corporate materials. The symbol must be in the logo's designated color and positioned as per original specifications.

**Notes:**

- Co-branding for garments includes ONTHEGO Icon on custom products such as polos, hoodies, jackets, tees, etc. The logo color should follow Artwork + Garment branding and core brand colors.
