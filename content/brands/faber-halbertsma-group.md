## Brand Guidelines

### 2020

**Logo Information:**

- Examples of misapplication of the logo: Don't use colors outside branding palette, don't use a different font, don't make it extra bold, don't place the words vertically beneath each other, don't make it italic, don't apply an angle, don't use a contour, don't apply shadows.

**Color Information:**

- Main colors: Green (CMYK: 96 / 22 / 63 / 0; RGB: 0 / 135 / 116; HEX: #008A75; Pantone Coated: Pantone 569 C; Pantone Uncoated: Pantone 334 U; RAL: 6016) and Dark Grey (CMYK: 0 / 0 / 0 / 85; RGB: 74 / 74 / 73; HEX: #494848; Pantone Coated: Pantone 7540 C; Pantone Uncoated: Pantone 433 U; RAL: 7022).
- Secondary colors: Light Blue (CMYK: 65 / 0 / 10 / 5; RGB: 64 / 171 / 208; HEX: #40abd0; Pantone Coated: Pantone 637 C; Pantone Uncoated: Pantone 637 U), Pink (CMYK: 8 / 70 / 15 / 0; RGB: 210 / 83 / 133; HEX: #d25385; Pantone Coated: Pantone 709 C; Pantone Uncoated: Pantone 1915 U), Orange (CMYK: 0 / 30 / 90 / 0; RGB: 246 / 172 / 0; HEX: #f6ac00; Pantone Coated: Pantone 137 C; Pantone Uncoated: Pantone 123 U), Green-Yellow (CMYK: 40 / 0 / 100 / 0; RGB: 163 / 193 / 0; HEX: #a3c100; Pantone Coated: Pantone 382 C; Pantone Uncoated: Pantone 381 U).
- Contrast checker defines how different colors (white, dark grey, pink, green, orange, blue) are used to enhance text readability at font sizes over and below 18pt or 14pt.

**Typography Information:**

- Typography for print: Headlines, sub-headlines, quotes, and streamers use 'Khand' font with weights Light, Regular, Medium, Semibold, and Bold. Body text uses 'Roboto Condensed' font with weights Light, Light Italic, Italic, Regular, and Bold.
- Online typography (Desktop): Header 1 (Khand Semibold, 70px, Line-height: 74px), Header 2 (Khand Semibold, 42px, Line-height: 52px), Header 3 (Khand Semibold, 26px, Line-height: 36px), Header 4 (Roboto Bold, 18px, Line-height: 28px), Body intro (Roboto Bold, 22px, Line-height: 32px), Body text (Roboto Regular, 16px, Line-height: 26px).
- Online typography (Mobile): Header 1 (Khand Semibold, 42px), Header 2 (Khand Semibold, 32px), Header 3 (Khand Semibold, 22px), Header 4 (Roboto Bold, 16px), Body intro (Roboto Bold, 18px), Body text (Roboto Regular, 16px).
- Calibri font family (available weights: Regular, Bold, Italic, Bold Italic) is used in fallback cases for online tools or corporate stationery.

**Imagery Photography:**

- Illustration usage: Graphics should be accessible and complementary to the main colors (green and dark grey). A specific illustration style has been developed to visually explain services and products.
- Isometric perspective is used for illustrations, following a 30-degree angle.

**Spacing Layout:**

- Specific templates available for items such as business cards (85 x 55mm), A5 envelopes, A4 letter sheets, tables, Word templates, and PowerPoint templates. Design elements are noted in the footer for themes.

**Notes:**

- The brand guideline document mentions central IT & MarCom controls over employee email signatures via Exclaimer.
