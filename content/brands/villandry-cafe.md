## Brand Guidelines

### 2012

**Logo Information:**

- The logo combines the name 'Villandry' and a descriptive strapline 'Grand Café'. The name is set in Fairplex Narrow Book, customized for ornamental French Brasserie lettering. Strapline 'Grand Café' uses Bell Gothic BT Roman and is slightly curved for French brasserie reference.
- Authorized logo versions: Large logotype for widths >50mm; Small logotype for widths <50mm; Digital logotype for online use at any size.
- Logotype free zone equals the height of the name 'Villandry'. This space must not contain text, drawings, or photographs and enhances appearance and correct positioning.
- Logo should always be centered in designs/formats, horizontally and vertically. Prioritize free zone space in positioning.
- Logotype can be used in 3 Villandry frame shapes if the format (e.g., sticker, tag) matches those shapes. Position centered and respect free zone.
- Do not alter authorized versions of the logotype (e.g., redraw or modify). It must only be reproduced using official files provided on CD.

**Color Information:**

- Villandry color palette includes: Two logotype colors (warm grey and black) and three background colors (orange, blue, yellow). Background colors must not combine.
- Color specifications: 
Orange: Pantone 164U; CMYK 0/53/63/0; RGB 255/143/99 
Blue: Pantone 310U; CMYK 48/0/9/0; RGB 120/209/227 
Yellow: Pantone 107U; CMYK 2/4/80/0; RGB 255/232/82 
Warm grey: Pantone 7531U; CMYK 45/42/57/10; RGB 139/129/110 
Black: Pantone Black U; CMYK 0/0/0/100; RGB 0/0/0
- Logotype on colored background must be black; logotype on white background must be warm grey.
- Colors should not be used randomly or purely decoratively. Background colors must adhere to guidelines for individual usage.

**Typography Information:**

- Authorized typefaces: Fairplex Narrow Book (used in lowercase only) and Bell Gothic BT Roman (capitalized permitted).
- Headlines should follow a clear hierarchy. Excessive font weights or sizes to emphasize text sections must be avoided.
- Headline example: Fairplex Narrow Book, Font Size: 16pt, Leading: 18pt, Tracking: 15pt. Body text example: Fairplex Narrow Book, Font Size: 12.5pt, Leading: 16pt, Tracking: 15pt.
- Typography rules: All text must be flush left with 2pt additional line spacing. Poorly set text harms overall identity.
