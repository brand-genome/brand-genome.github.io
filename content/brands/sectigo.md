## Brand Guidelines

### 2018

**Logo Information:**

- Sectigo logotype leverages the Rajdhani font with customized 'S' and 'G' letterforms. The logo mark should not be used in isolation without the accompanying logotype.
- Logo variations include: knocked-out logo with green 'S' used on PANTONE 296 C, logo over light gray background, logo over images with whitespace, all-black and all-white logos.
- Do's and Don'ts for logo usage: Do not stretch, rotate, swap colors, use gradients or drop shadows, use unapproved colors, add text, or place over busy backgrounds.
- Clear space requirement: Minimum breathing room maintained around logo. Minimum size requirement: 1 inch wide.
- Transition logo guidelines: 'Formerly Comodo CA' tagline to be used to assist rebrand recognition.
- Tagline art treated like a logo, with its own design element for consistency. Acceptable typeface for tagline in body text: Roboto or Arial.
- Partner logos include variations such as 'Powered by Sectigo'.
- Old logo vs. new logo comparison provided.

**Color Information:**

- Primary Colors: PANTONE 296 C (#041C2C, RGB: 9-27-42, CMYK: 91-76-55-68), PANTONE 2250 C (#00B373, RGB: 0-176-115, CMYK: 84-0-75-0). Secondary Colors: PANTONE 121 C (#FDD757, RGB: 254-214-85, CMYK: 1-14-78-0), PANTONE 292 C (#69B3E7, RGB: 103-177-226, CMYK: 56-16-0-0), PANTONE 143 C (#F1B434, RGB: 244-179-51, CMYK: 3-32-31-0).
- Guidelines for hierarchy, balance, and harmony in color usage: Primary colors should dominate communication, with appropriate use of secondary colors and negative space.

**Typography Information:**

- Typography guidelines specify Roboto font for external-facing materials (e.g., website, ads). Roboto weights: Light, Regular, Medium, Bold.
- Arial is used for corporate internal-facing materials (e.g., letterhead, emails, memos). Arial weights: Regular, Bold.
- Text styling examples provided for type hierarchy: titles, paragraphs, descriptors, etc. Specifications include font weight, leading, and case usage across examples.

**Imagery Photography:**

- Guidelines specify images should use dramatic negative space, unique angles, and aspirational compositions. Photography is categorized as authentic bright office spaces, ambitious subjects, and unique dramatic angles.
- Color adjustments recommended for images to align with Sectigo green/blue/gray tonal palette using photo editing software.

**Spacing Layout:**

- Grid rules for creating new icons: built on 32x32 pixel grid with 2-pixel stroke and buffer. No crossing buffer unless necessary.
- Icon scaling guidelines specify stroke thickness rules relative to size. Icons should communicate messages visually while preserving appeal.
- Tables and Graphs: Use Sectigo fonts and colors, keeping designs clean and organized.

**Brand Voice:**

- Grammatical rules include using Oxford commas, spacing conventions, and product name capitalizations. Examples provided throughout guidelines.

**Notes:**

- Sectigo emphasizes visual communication through a branded icon library using specific primary colors. Custom-drawn icons highlight web security topics.
- Negative space treated as an important design element, emphasizing minimalistic yet visually impactful communication.
