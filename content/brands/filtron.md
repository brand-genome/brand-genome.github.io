## Brand Guidelines

### 2019

**Logo Information:**

- The FILTRON logo is a registered trademark, requiring consultation with authorized employees of MANN + HUMMEL FT Poland Sp. z o.o. Logo color guidance: CMYK 29 0 36 100 and RGB 28 28 28 for light backgrounds; CMYK 0 0 0 0 and RGB 255 255 255 for dark backgrounds.
- The minimum protection area around the logo equals the height and width of the letter 'O'.
- Recommended to use a black logo on light backgrounds and a white logo on dark backgrounds.
- Incorrect usages of the FILTRON logo: 1) Using colors other than those specified. 2) Modifying the design (e.g., stretching, changing typography, removing elements). 3) Incorrect filling of the letter 'O'. 4) Using the previous slogan ('The Original'). 5) Applying additional background.
- Logo centering: The length of the ® symbol is not included when determining the logo's length for centering purposes. Proper and improper central axes are illustrated.

**Color Information:**

- Background colors allowed: F1 (CMYK: 12 8 8 0, RGB: 228 230 231, Pantone 429 C 30%), F2 (CMYK: 0 0 0 5, RGB: 246 246 246), F3 (CMYK: 22 15 17 0, RGB: 221 223 225, Pantone 427 C). F3 should be used only with F2 for large-format prints (posters, banners, etc.).
- Text colors: Basic Text Color (CMYK: 0 0 0 100, RGB: 0 0 0), and Accent Color (CMYK: 100 53 2 16, RGB: 0 76 151, Pantone 2945C). Text areas using the accent color as the background must have white text.
- Filter packaging colors: Fuel Filters (CMYK: 50 0 100 0, RGB: 150 190 25, Pantone 376 C), Oil Filters (CMYK: 0 10 100 0, RGB: 255 220 0, Pantone 109 C), Air Filters (CMYK: 0 100 100 0, RGB: 255 0 0, Pantone 186 C), Cabin Filters (CMYK: 75 15 0 0, RGB: 40 170 225, Pantone 2985C).

**Typography Information:**

- Primary font: FUTURA. Styles: FUTURA Bold (used for headlines and titles) and FUTURA Book (used for other content).
- For digital materials (e.g., web pages, HTML5 banners), use fonts from the Google Fonts library tailored to the design and approved by the Filtron Marketing Department.
- In Microsoft Office documents, TAHOMA or ARIAL is acceptable.
