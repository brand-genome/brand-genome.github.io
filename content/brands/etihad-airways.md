### 2016

**Logo Information:**

- Arabic and language variations for international markets with local-language integration (Chinese, Korean, Japanese, Thai). Note that only Arabic is replaced; 'Abu Dhabi' remains in English. Variations are created by Visual Communications.
- Clear space rules using the 'D' rule derived from the letter 'D' in the ETIHAD wordmark. Minimum size indicated for applications to preserve legibility.
- Incorrect logo usages specified (e.g., distortion, stretching, resizing, applying effects, altering colors, setting against faded or busy backgrounds). Historical logo versions discontinued and must be replaced; Visual Communications should be informed if such occurrences are found.

**Color Information:**

- Two gold colors defined: Etihad Bright Gold (for dark backgrounds) and Etihad Gold (for pearl/lighter backgrounds). CMYK values provided for both.
- Standard and premium logo printing. Premium application uses PANTONE 8640C metallic; CMYK values differ based on background color. Black-and-white usage limited to specific applications like newspapers; requires 100% black reproduction.
- Third-party/joint communication guidelines for logo usage on photographic or brand-specific backgrounds. Logos should use ETIHAD Gold or White for optimal visibility.

**Typography Information:**

- Etihad Altis typeface explicitly described as part of the brand identity. Includes weights (Thin, Light, Book, Text, Medium, Bold, Extra Bold), italics versions, and usage specifics. Trebuchet MS used as a digital substitute only when Etihad Altis isn't supported, such as in CMS or third-party documents.
- Arabic typeface DIN Next for Etihad specified with weights (Light, Book, Medium). Arabic numerals to be used, avoiding Hindi numerals.

**Imagery Photography:**

- Facets of Abu Dhabi used in new aircraft exteriors and interiors, connecting facets into larger canvases.

- Golden Facets application for visually significant items like packaging or premium brochures; approvals required.
