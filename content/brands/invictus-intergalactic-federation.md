We have a strict scale format that we    
require everyone to follow.
The logo scales down in halves, down 
to a 1/8th. Please reference this scale 
for all further scales™ in this guide.

and can be used for reference. The 
larger ﬁPﬂ is half the size of the logo 
and should only be used for main titles 
or partnership names.
The smaller ﬁpﬂ is a quarter of the full 
logo size. This is to be used as sub-text, 
usually placed below the main title/text. 
The only exception is the full organiza
-
tion name.

- Logo | Partner
lock-up scale
For horizontal logos and word-mark
