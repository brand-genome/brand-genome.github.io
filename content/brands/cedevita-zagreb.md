## Brand Guidelines

### 2016

**Logo Information:**

- Daily logo presentation: Main support of the new club identity, includes positive (color and black/white) and negative versions (color and black/white) for different backgrounds.
- Negative version adjusts the black outline of the C and colors (orange outline and black spin).
- Minimum size requirements for the Daily Logo: Minimum size 10mm, with minimum safety margins defined as 4X (where X is the vertical height of the logo).
- Official logo presentation includes three elements (C, Cedevita, Zagreb) used together as one block. Negative and positive versions for different backgrounds are specified.
- Minimum size and proportions for the official logo: Minimum size of 15mm and minimum protection margins of 4X.
- Alternative logo presentation: Used for derivatives like shirts and caps but not for official use. Negative version specified.
- File format or explicit file-related mentions not provided in the document.

**Color Information:**

- Primary color is Orange: Pantone 1505 C / CMYK 0/60/90/0 / RGB 255/110/0 / HEX #FF6E00.
- Secondary bright and modern color tones recommended: Pantone 387 C, Pantone Yellow 012 C, Pantone 382 C, Pantone 368 C, Pantone 604 C, Pantone 7465 C, Pantone 306 C. Hex, RGB, and CMYK values defined for each.

**Typography Information:**

- Two main typefaces: DIN Next LT Pro and Museo. DIN Next LT Pro is used for detailed text like dates, hashtags, and body text, while Museo is for titles and claims. Colors: Orange, Black, or White.
- Font styles based on role: Museo 900 for main titles, Museo 500 for subtitles, DIN Next LT Pro Condensed (Heavy, Bold, Medium, Regular) for dates, hashtags, etc., and DIN Next LT Pro Regular/Bold for body text.

**Imagery Photography:**

- Ball element with a spin can be used as an independent identity element but must be accompanied with the Daily or Official Logo. The ball proportionally cannot be bigger than the logo.

**Spacing Layout:**

- Minimum spacing/safe zones: Daily Logo requires 4X space (X = height of C in logo). Official Logo also requires 4X protection margins.
