## Brand Guidelines

### 2017

**Logo Information:**

- Word of Life logos must retain consistency in orientation, color, and style. Alterations such as taking apart, changing components, or disassembling elements are prohibited.
- Approved primary Word of Life logo orientations include Horizontal Version, Vertical Version, Alternate Version (for narrow layouts), and Mark Alone (icon-only).
- Logo colors (CMYK): 73, 0, 0, 73; 48, 0, 0, 0; 0, 0, 0, 90; 0, 0, 0, 50.
- Logo typefaces: Helvetica Bold and Helvetica Light.
- Standard and reversed color formats allowed. Standard includes full-color logos appearing on white, light, or neutral backgrounds, while reversed uses a white logo on dark backgrounds.
- Clear space rules: Logos must be surrounded by clear space equivalent to the height and width of the logomark on all sides.
- Minimum size requirements: Full logo text ('Word of Life') must not be smaller than 1 inch wide and 1/8 inch tall; icon-only version should not be smaller than 1/4 inch in both width and height.
- Prohibited logo treatments: Adding drop shadows, strokes, or bevels; rearranging or stretching logo elements; and modifying the fonts.
- Sub-brand logos follow the same guidelines for approved orientations, color formats, clear space, and size requirements.

**Color Information:**

- Primary Brand Colors: Dark Blue (CMYK: 73, 0, 0, 73; HEX: #004F68; RGB: 0, 79, 104), Light Blue (CMYK: 48, 0, 0, 0; HEX: #73D1F5; RGB: 115, 209, 245), Charcoal Gray/Black (CMYK: 0, 0, 0, 90; HEX: #414042; RGB: 65, 64, 66), Medium Gray (CMYK: 0, 0, 0, 50; HEX: #939598; RGB: 147, 149, 152), Light Gray (CMYK: 0, 0, 0, 20; HEX: #D1D4D3; RGB: 209, 212, 211).
- Sub-brand colors: Bible Institutes (CMYK: 73, 0, 0, 42; HEX: #007EA1), Youth & Family Camps (CMYK: 48, 0, 0, 0; HEX: #73D1F5), Local Church Ministries (CMYK: 16, 97, 84, 6; HEX: #C42B36), International Ministries (CMYK: 6, 47, 94, 0; HEX: #E99531), Events (CMYK: 71, 1, 40, 0; HEX: #36BAAC).

**Typography Information:**

- Primary font: Helvetica (used for headlines, body copy, captions, etc.). Default Helvetica replacement: Nimbus Sans. Approved weights: UltraLight, Thin, Light, Regular, Bold, Black (Neue, Condensed, Oblique, Italic allowed). Prohibited: Helvetica Rounded.
- Secondary font: Merriweather (used for extended body text or additional variety); Replacement: Georgia; Available weights: Light, Regular, Bold, Ultra Bold.
- Web font replacement for Helvetica: Roboto

**Imagery Photography:**

- Photography reflects Word of Life's dynamic experiences and history, and can be styled using full-color or Color Overlays.
- Color overlay instructions: Convert photos to black-and-white with 70% opacity color overlays in Adobe software using the 'normal' layer style; overlays may use primary or sub-brand colors.

**Spacing Layout:**

- Logos must have clear space equivalent to the height and width of the logomark on every side.

**Brand Voice:**

- Brand promise emphasizes creating a cohesive and effective visual brand experience. Values include creating faith-defining experiences, outreach, life-changing experiences, and making an eternal impact.
- Positioning statement: 'Word of Life creates faith-defining experiences that give students and families the opportunity to encounter God and grow in their spiritual walks.'

**Notes:**

- Key guidelines for apparel include specific logo placements: t-shirts (tagline on back collar), hoodies (sleeve, shoulder, or near wrist), shorts (side, bottom left seam), pants (near pocket or side seam), hats (back, bottom, or above snapback).
- Graphic elements include Accent Ribbons, Color Overlays, Grunge Textures (Youth & Family Camps), Wood Textures, and Sub-brand Icons. Ribbons are used as emphasis or breaks, while overlays and textures apply unique visual effects.
