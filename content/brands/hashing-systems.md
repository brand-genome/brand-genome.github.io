## Brand Guidelines

### 2019

**Logo Information:**

- Hashing System logo is the combination of the company name and icon. It includes a primary logo, a stacked logo format, and an icon that can be used independently. The primary logo combines the logomark and wordmark.
- The logo should not be altered, re-proportioned, or modified in any way. Logomark color: Turquoise (#07E4B7), Black (#333333). Wordmark uses the Poppins Medium font.
- Reversed logo version is used when the logo appears as white on a solid dark background.
- The minimum logo size for readability: 94 pixels/25 mm/1 inch for displays of 72 DPI or higher; 38 pixels/10 mm/0.4 inch for smaller usages.
- Clear space for the primary logo is equivalent to the height of the logo while for the secondary logo, it is 1/2 the height of the logo.
- Gray-scale and monochromatic logos (black, white, or gray) are used in faxes, engraving, embroidery, or collaborations to minimize visual clutter.
- Do’s and Don’ts: Do not change colors, outline, rotate, stretch, condense, or modify the logo. Ensure contrast between the logo and background and avoid cropping the logo.

**Color Information:**

- Primary color palette: Turquoise (#07E4B7, CMYK: 63, 0, 43, 0, Pantone 3265 C), Navy (#1B2D44, CMYK: 96, 77, 45, 49, Pantone 289 C), Dark purple (#211944, CMYK: 100, 100, 38, 44, Pantone 2765 C), Black (#363636, CMYK: 68, 59, 56, 64, Pantone 152 C), and Blue (#00BCFF, CMYK: 67, 7, 0, 0, Pantone 2995 C).
- Secondary colors: Dark Orange (#FF6700, CMYK: 0, 70, 94, 0, Pantone 717 C); Purple (#7F55DE, CMYK: 72, 72, 0, 0, Pantone 426 C), Green (#00BD61, CMYK: 73, 0, 77, 0, Pantone 7481 C), Orange (#F1A402, CMYK: 3, 40, 97, 0, Pantone 151 C), and Red (#FE2D35, CMYK: 0, 89, 73, 0, Pantone 1788 C).

**Typography Information:**

- Primary font: 'Poppins', used for the logo wordmark, headings, and subheadings. Poppins is described as a clean and contemporary sans-serif typeface.
- Secondary font for body text: 'Verdana', described as a humanist sans-serif typeface developed by Matthew Carter for Microsoft Corporation.
- Sample typography hierarchy: headings use Poppins and are colored #000255; subheadings use Poppins with a size of 24pt and colored #000000; body text uses Verdana, size 11pt and colored #000000.

**Spacing Layout:**

- Primary logo clear space is equivalent to the height of the logo; secondary logo clear space is 1/2 the height of the logo. Favicon clear space must also respect 1/2 of its height.

**Notes:**

- The brand guidelines date back to October 2019 and are created by Hashing Systems, Inc in Dallas, TX.
